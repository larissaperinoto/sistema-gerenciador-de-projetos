## Sistema gerenciador de projetos

##### Tecnologias

- Node.js
- Express
- React.js
- MySQL
- Docker
- Sequelize

#### Pré requisitos para rodar o projeto

- Docker
- Node.js (>= 18.16)

#### Rodando o projeto localmente

Clone o repositório

      git clone git@github.com:larissaperinoto/sistema-gerenciador-de-projetos.git

Inicie os containers docker

      docker-compose up
